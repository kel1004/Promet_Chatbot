## Promet_Chatbot
# Chatbot school project for Pamantasan ng Lungsod ng Marikina
*Date of Creation: 03/03/2026*

I am an alumni of PLMar and currently enrolled on Technological University of the Philippines - Manila. This chatbot was made as a project requirement to our related course subject. It was built using: HTML as frontend, CSS for styles, and Vercel for both hosting and backend development.

Feel free to use this chatbot as a reference.
