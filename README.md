## Promet_Chatbot
# Chatbot school project for Pamantasan ng Lungsod ng Marikina
*Date of Creation: 03/03/2026*       

I am an alumni of PLMar and currently enrolled on Technological University of the Philippines. This chatbot was made as a project requirement to our related course subject. It was built using: HTML as frontend, CSS for styles, JS for functional logic, and Vercel for both hosting and backend development(serverless functions).

Feel free to use this chatbot as a reference.

(Note: This is powered by Google Gemini 2.0 Flash API, which is a free tier API.)
